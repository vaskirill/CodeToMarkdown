# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Professional GitHub repository structure
- Comprehensive documentation
- CI/CD pipeline with GitHub Actions
- Security and contribution guidelines

## [1.2.0] - 2025-01-19

### Added
- MIT License
- Copyright headers in all source files
- Optimized file discovery algorithms
- Improved markdown generation performance
- Enhanced error handling and logging

### Changed
- Refactored code structure for better maintainability
- Optimized memory usage in file processing
- Improved UI responsiveness
- Updated dependencies

### Fixed
- Fixed file size calculation issues
- Resolved encoding problems with various file types
- Fixed memory leaks in large project processing
- Improved error handling for corrupted files

### Removed
- Unused code and dependencies
- Redundant file processing steps
- Obsolete configuration options

## [1.1.0] - 2025-01-15

### Added
- Preview functionality
- Enhanced file filtering
- Improved UI design
- Better error messages

### Changed
- Updated PySide6 to latest version
- Improved file discovery performance
- Enhanced markdown formatting

## [1.0.0] - 2025-01-10

### Added
- Initial release
- GUI application with PySide6
- File discovery and filtering
- Markdown generation
- Support for multiple programming languages
- .gitignore support
- Configuration management
- Internationalization (Russian/English)
